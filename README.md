# rndimage: random JPEG image generator

`rndimage` generates JPEG of a given size filled with randomly-colored pixels written in Go.

## Getting Started

### Installing

```
go get github.com/svlapin/rndimage
```

### Generate image

```
rndimage 640 480 > test_640x480.jpg
```
